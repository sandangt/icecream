# Documentation of Icecream e-commerce

## vscode files (`.vscode`)

    ```json
    {
      "recommendations": [
        "hediet.vscode-drawio",
        "pomdtr.excalidraw-editor"
      ]
    }
    ```
  - `settings.json`
    ```json
    {
      "excalidraw.image": {
        "exportScale": 1,
        "exportWithBackground": true,
        "exportWithDarkMode": false
      },
      "excalidraw.theme": "dark",
      "excalidraw.workspaceLibraryPath": "libs/main.excalidrawlib",
      "hediet.vscode-drawio.theme": "min",
      "hediet.vscode-drawio.appearance": "light"
    }
    ```
